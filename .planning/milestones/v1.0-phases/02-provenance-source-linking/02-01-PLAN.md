<!-- The exact API to build, transcribed from RESEARCH "Trust Module API Design". Build to this contract. -->

js/trust/index.js must export THREE functions:

provenanceFor(input, ctx = {}) -> { tag: 'observed'|'estimated'|'unknown', source?: {label, url} }
  - input.marketcap === true            -> { tag:'observed', source:{label:'companiesmarketcap.com', url: ctx.meta?.source || ctx.meta?.top100_source_url} }  (or {tag:'unknown'} if neither url)
  - confidence string = String(input?.confidence ?? input?.cf ?? '').toLowerCase()
      startsWith('high')   -> tag = 'observed'
      startsWith('medium') -> tag = 'estimated'
      else                 -> tag = 'unknown'
  - source FK = input?.sourceId ?? input?.sf ; src = (fk && ctx.sourceIndex) ? ctx.sourceIndex[fk] : null
      if (src && src.url) -> attach source:{label: src.title || src.id || 'Source', url: src.url}
      else                -> NO source key (dangling FK / empty sf / absent -> tag only)
  PURE: no DOM, no window, no STATE access. Fully importable + callable in Node.

badgeHtml(prov) -> string   (PURE — returns the HTML string; this is what tests assert against)
  - label: observed->'Observed', estimated->'Estimated', unknown->'Unknown'
  - cls:   observed->'confidence-high', estimated->'confidence-medium', unknown->'confidence-low'
  - emits <span class="prov-badge confidence-badge {cls}" role="img" aria-label="Provenance: {label}{, source available|, no source}" title="...">{label}</span>
  - when prov.source: append <a href="{url}" target="_blank" rel="noopener noreferrer" class="source-link">source ↗</a>
  - MUST escape source.title and source.url (reverse-tabnabbing + no-XSS posture, RESEARCH Security Domain).
    Replicate the existing escapeHtml from js/data/index.js:9 locally (trust must stay dependency-light / DOM-free) OR import it — your discretion, but do not pull DOM in.
  - Only emit the <a> when url starts with 'http' (RESEARCH V5 control).

renderProvenanceBadge(el, prov)  -> void
  - if (!el) return;  el.innerHTML = badgeHtml(prov);   (the ONLY DOM touch in the module)

Existing escapeHtml to mirror (js/data/index.js:9):
  String(input||"").replace(/&/g,"&amp;").replace(/</g,"&lt;").replace(/>/g,"&gt;").replace(/\"/g,"&quot;").replace(/'/g,"&#39;")
